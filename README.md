# App Privacy Policies Modern

This repository contains modern responsive privacy policy pages for 109 apps.

## GitHub Pages setup

1. Create a GitHub repository named `app-privacy-policies`.
2. Upload all files from this folder to the repository root.
3. Go to Settings > Pages.
4. Select Deploy from a branch.
5. Select branch main and folder /root.
6. Save.

Main URL example:

`https://YOUR_GITHUB_USERNAME.github.io/app-privacy-policies/`

App URL example:

`https://YOUR_GITHUB_USERNAME.github.io/app-privacy-policies/apps/visinews-iptv/`
